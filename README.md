# 1960·70년대 신문·잡지를 인용하는 학술논문 분석

KCI 전문(全文) 코퍼스 **137,961편**을 전수 스캔해, **각주에서 1960·70년대 신문·잡지를 2건 이상 인용한 논문 8,066편**을 '관련 논문'으로 정의하고 이들의 신문·잡지 인용을 집계한 인터랙티브 대시보드입니다.

🔗 **공개 페이지**: https://gkman1-art.github.io/kci-periodical-citation/

## 담긴 내용
- 신문 58,211건(586종)·잡지 13,251건(27종) 인용의 규모·집중도(동아·경향·조선 72.3%)
- **인용된 기사·글의 월별 발행 시점 분포**(1960.01–1979.12) — 라인차트 + 연×월 히트맵 (정점이 4·19(1960.5)·5·16(1961.5) 등 정치사 결절점과 일치)
- **3대 종합지 비교**: 사상계 · 세대 · 신동아 (시기 프로파일·인용 형식·기능 대비 + 매체별 심층)
- 분야별 인용 강도(역사학→신문 / 문학→잡지), 매체를 폭넓게 동원한 논문(저자 포함)
- 방법·정밀도 한계(적대적 감사, 카테고리별 72~85%)

## 방법 요약
전체 코퍼스를 전수 스캔해, 각주·미주·참고문헌에서 **발행 시점이 1960–1979년인 신문·잡지를 2건 이상 인용한 논문**을 관련 논문으로 정의했습니다(제목·키워드 표기에 의존하지 않아 '박정희·유신' 등 주제어 누락이 없음). 신문·잡지는 정규식으로 분류하고 60·70년대 발행분 인용만 집계했습니다. 빈도는 추출 레코드 기준 추정치이며, 발행처·2차문헌·동명이매체 혼입 등 정밀도 한계를 동반합니다(보고서 §7).
