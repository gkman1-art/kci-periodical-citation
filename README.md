# 학술논문의 신문·잡지 인용 지도 (1954–1979)

KCI 전문(全文) 코퍼스 **145,088편**을 전수 스캔해, **각주·참고문헌에서 1954–1979년 신문·잡지를 2건 이상 인용한 논문 9,796편**을 '관련 논문'으로 정의하고, 이들이 인용한 신문·잡지의 종류·규모·발행 시점·활용 양상을 집계한 인터랙티브 대시보드입니다.

🔗 **공개 페이지**: https://gkman1-art.github.io/kci-periodical-citation/

- 신문 83,794건(674종)·잡지 17,603건(28종) 인용의 규모·집중도(동아·경향·조선 72.6%)
- **인용된 기사·글의 발행 시점 분포**(1954.01–1979.12) — 월별/연별 전환 + 연×월 히트맵 (정점이 4·19(1960.5)·5·16(1961.5) 등 정치사 결절점과 일치)
- **주요 종합지 비교**: 사상계·세대·신동아·창작과비평·현대문학·청맥 (시기 프로파일·인용 형식·기능 대비)
- 분야별 인용 강도(역사학→신문 / 문학→잡지), 매체를 폭넓게 동원한 논문(저자 포함)
- 방법·정밀도 한계(적대적 감사, 카테고리별 72~85%)

## 방법 요약
전체 코퍼스를 전수 스캔해, 각주·미주·참고문헌에서 **발행 시점이 1954–1979년인 신문·잡지를 2건 이상 인용한 논문**을 관련 논문으로 정의했습니다(제목·키워드 표기에 의존하지 않아 '박정희·유신' 등 주제어 누락이 없음). 신문·잡지는 정규식으로 분류하고 1954–79년 발행분 인용만 집계했습니다. 빈도는 추출 레코드 기준 추정치이며, 발행처·2차문헌·동명이매체 혼입 등 정밀도 한계를 동반합니다(보고서 참조).

## 대학사 연구는 무엇을 사료로 쓰는가 (univ.html)

1954~1979년의 한국 대학을 다룬 논문 47편의 본문·각주를 전수 검토해 사료 사용을 측정하고, 같은 시기 일간지와 대학신문의 지면을 맞대어 본 결과. 탐색 범위는 신문·잡지 인용 지도와 같은 구간이다.

- 일간지를 사료로 인용한 논문 74.5%, 대학이 스스로 발행한 매체를 인용한 논문 57.4%. 한쪽만 택한 경우 일간지 13편 대 대학 매체 5편
- 일간지 3종이 1954~1979년 제목에서 호명한 대학: 서울 10곳 12,755건 대 지방 10곳 1,680건(7.6배)
- 타 대학을 부른 정도: 『부대신문』 8.99‰ 대 『대학신문』 0.87‰(10.3배)
- 두 신문 필자 8,795명 중 68%가 생애 단 한 편만 기고. 『부대신문』은 82%
