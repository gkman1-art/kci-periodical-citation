# 1960·70년대 학술논문의 신문·잡지 인용 분석

KCI 전문(全文) 코퍼스 **137,961편**의 각주·참고문헌에서, 1960·70년대를 다룬 논문 **2,410편**이 인용한 신문·잡지를 추출·집계한 인터랙티브 대시보드입니다.

🔗 **공개 페이지**: https://gkman1-art.github.io/kci-periodical-citation/

## 담긴 내용
- 신문 35,286건(395종)·잡지 7,106건(23종) 인용의 규모·집중도·피인용 시점
- **인용된 기사·글의 월별 발행 시점 분포**(1960.01–1979.12) — 라인차트 + 연×월 히트맵 (정점이 4·19·5·16·한일협정 등 정치사 결절점과 일치)
- **3대 종합지 비교**: 사상계 · 세대 · 신동아 (시기 프로파일·인용 형식·기능 대비 + 매체별 심층)
- 분야별 인용 강도(역사학→신문 / 문학→잡지), 매체를 폭넓게 동원한 논문(저자 포함)
- 방법·정밀도 한계(적대적 감사, 카테고리별 72~85%)
